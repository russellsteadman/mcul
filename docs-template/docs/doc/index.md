# Documentation

## Overview

The `mcul` package exports classes [`Molecule`](/doc/molecule/) and [`Atom`](/doc/atom/).

```js
// JS import
import { Molecule, Atom } from 'mcul';

// Node.js
const { Molecule, Atom } = require('mcul');

// via script tag
const { Molecule, Atom } = window.mcul;
```
